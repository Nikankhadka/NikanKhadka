**GitHub:** https://github.com/Nikankhadka/fyp

MeroGhar is an Airbnb-style rental management platform built for Nepal's rental/property market.

**Key features:**

- Tenant, property owner, and admin roles
- Property listings, dashboards, booking, check-in/check-out
- Real KYC workflow, phone/email verification, and social login
- PayPal and Khalti payment integration
- Zoho Mail notification support
- Verified reviews and role-based workflows
- Dockerised monorepo structure
- GitHub Actions CI/CD
- Playwright testing
